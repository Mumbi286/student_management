# Project name 
 Student management system

# Project overview
This is a python based programmed project that is mostly used in the school institution. This system enables them record and managed there data using the SQL and OOP principles.

## Features
- Adds new students
- View all students
- Update student details
- Delete student records
- Input validation with proper error messages
- User friendly and easy to use on the CLI

# Installation requirements
1. pipenv install
- Installs the dependencies
2. pipenv shell
- To access the virtual environment 
3. pip install click

# How to run the project using on CLI
- To add a student
    python3 main.py add
- To view students
    python3 main.py view
- To update a student
    python3 main.py update
